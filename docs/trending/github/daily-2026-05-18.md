## 观察

- 今日榜单的主线不是单一模型或框架，而是“让 AI 更能接入工作流”：个人 Agent、CLI 工具封装、Agent Skills、OSINT 看板、语音 Agent、生产级 Agent 教程同时出现。
- Agent Skills 相关项目明显升温：`tech-leads-club/agent-skills` 和 `K-Dense-AI/scientific-agent-skills` 都进入 Top 10，说明“可复用指令 + 工具说明 + 工作流模板”正在被当作独立资产传播。
- 本地和自托管叙事很强：`openhuman`、`Open-Generative-AI`、`Shadowbroker`、`dograh`、`cal.diy` 等都强调本地、私有、自托管或可控基础设施。
- 成熟基础设施仍有强势表现：`oven-sh/bun` 以 910 stars today 排在当日新增关注前列，说明非 AI 项目只要基础设施价值足够明确，仍能在 AI 热潮中保持可见度。
- 生成式媒体和语音 Agent 正在工具化：`Open-Generative-AI` 和 `dograh` 都不只是模型 demo，而是提供可运行的产品界面、桌面包或 Docker 启动路径。
